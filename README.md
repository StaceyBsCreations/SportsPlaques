# Stacey B’s Creations – Custom Sports Plaques

Static GitHub Pages site for collecting order details for:
- Plaques (layered wood, framed look)
- Stand-ups
- Stadium Series Pennants

## Folder structure

- index.html
- styles.css
- options.js  (dropdown options; must load before script.js)
- script.js   (UI wiring / preview / cart / email)
- assets/     (logo + future site assets)
- examples/   (product photos; we’ll wire into the gallery)
